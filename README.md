# amazon-cancellation-analysis
Analysis of Amazon order cancellations using Python and pandas

Amazon Order Cancellation Analysis

Цель проекта:
анализ отмен заказов и выявление категорий с повышенным уровнем отмен.

Инструменты:
Python
pandas

Что было сделано:

- рассчитан общий процент отмен
- рассчитан процент отмен по категориям
- определена категория с самым высоким процентом отмен
- рассчитана средняя сумма отменённого заказа
- приведены бизнес-выводы и рекомендации по категориям
- выявлены проблемные категории

Ключевые результаты:

Общий процент отмен: 14.14%

Категории с наибольшими потерями:
Set — 526,844 INR
Kurta — 307,800 INR

Категория с самым высоким процентом отмен:
Saree — 17.95%

Файлы:

amazon.csv — исходный датасет  
amazon.py — код анализа  
analysis.xlsx - сводная таблица по категориям с ключевыми числовыми данными
